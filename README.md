> **Turn natural language into responsive websites in seconds.**

**Chat 2 Code** is a generative AI tool that enables users to build, style, and iterate on web pages using simple English prompts. It features a real-time **split-screen interface** where users chat with an AI agent on one side and see their code render live on the other.
